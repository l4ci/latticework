# blue-ocean-strategy

## What this is

Blue Ocean Strategy is about getting out of crowded markets where rivals fight over the same shrinking demand until margins bleed out ("red oceans") and creating uncontested market space instead ("blue oceans"). The engine is value innovation: pursuing differentiation and low cost at the same time rather than trading one for the other. You lift the value buyers get while you strip out the costly attributes the industry competes on but buyers no longer care about, and the cost saving funds the value lift. The framework comes from W. Chan Kim and Renée Mauborgne, set out in *Blue Ocean Strategy* (Harvard Business School Press, 2005) and built on their earlier Harvard Business Review work through the 1990s. Its core tools are the strategy canvas, the ERRC four-actions grid, and the six paths.

Value innovation, the strategy canvas, the ERRC grid, the six paths, and the focus/divergence/tagline test are in [references/blue-ocean.md](references/blue-ocean.md).

## What it does

The skill runs the analysis across three stages. First it maps the strategy canvas: the buyer-valued factors the industry competes on and how each player scores on them. Then four subagents work in parallel, one per action of the ERRC grid (eliminate, reduce, raise, create), each proposing concrete moves with the cost and value effect spelled out. A synthesis pass then assembles the new value curve, checks that the move actually cuts cost as well as lifting value, and tests it for focus, divergence, and a tagline. The output is a saved markdown report with the strategy canvas, the ERRC grid, the new value curve, and the risks the move carries.

The framework is easy to do half of, and half of it does not work: a move that only raises and creates is differentiation at higher cost, and a move that only eliminates and reduces is a discount play rivals match on price. The skill forces all four actions, checks the cost side at synthesis, and flags a me-too curve that tracks the rivals' rather than diverging from them.

## When to use it

Reach for Blue Ocean when the answer is to change the basis of competition rather than win the existing one: an industry has converged on the same factors and competes on price, or a new offering needs a position rivals are not already fighting over. The unit of analysis is a strategic move (an offering and the market it could create), not an industry or a company as a whole. It pairs with the structural tools rather than replacing them:

- If you first need to read the industry's profit structure to see whether it is worth reshaping, run [porters-five-forces](../porters-five-forces/).
- If you want a broad position read before deciding to reshape the market, run [swot-analysis](../swot-analysis/).
- If the goal is to sharpen the value a specific customer segment is buying, use [value-proposition-canvas](../value-proposition-canvas/).
